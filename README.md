## About

This page showcases **Quick Lofi**, a GNOME Shell extension that lets you play lofi and ambient sounds from the top bar.

No browser tabs.  
No heavy players.  
Just sound and focus.

## What Quick Lofi offers

• Minimal and distraction free experience  
• Native GNOME integration  
• Instant access from the top bar  
• Local files or online radios  
• Fully open source

## Features showcased

• One click play and pause  
• Volume control in the menu  
• Keyboard shortcuts  
• Drag and drop stations  
• Adjustable pop up size
